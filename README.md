# TripTrail Frontend

A React app for trip planning.

## Setup
1. `npm install`
2. Copy `.env.example` to `.env` and set `REACT_APP_API_BASE_URL`.
3. `npm start`

## Features
- User auth (login/signup)
- Trip creation, editing, deletion
- Daily plan viewing, activity add/remove
- Suggested activities, favorites
- Responsive UI matching mockups

## API Integration
Uses OpenAPI spec endpoints for real backend communication.
